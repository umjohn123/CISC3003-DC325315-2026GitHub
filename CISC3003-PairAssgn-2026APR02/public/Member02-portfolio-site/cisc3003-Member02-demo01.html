Spark‑based IT Job Market Analysis & Intelligent Recommendation

September – December 2025

Project Overview

A big data system that analyses large‑scale IT job postings using Apache Spark. It extracts insights about required skills, salary trends, and job locations, then provides personalised job recommendations based on user profiles.

Key Features

  • Data Processing: Processed 100,000+ job records with Spark (DataFrames, SQL, and MLlib).
  • Trend Analysis: Identified top in‑demand skills, salary distributions, and regional hotspots.
  • Recommendation Engine: Built a collaborative filtering model using Spark MLlib to suggest relevant jobs.
  • Dashboard: Created an interactive web dashboard using Flask and D3.js to visualise insights.

Technologies Used

Apache Spark Scala Python Flask D3.js MLlib

My Contributions

Developed the Spark ETL pipeline, implemented collaborative filtering for recommendations, and built the Flask backend. Also contributed to the front‑end visualisation with D3.js.
