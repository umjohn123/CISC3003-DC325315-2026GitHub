Medical Q&A System using LLM-Driven RAG

September – December 2025

Project Overview

A healthcare information system that leverages Large Language Models with Retrieval-Augmented Generation (RAG) to provide accurate answers to medical queries. The system retrieves relevant medical literature from a vector database and uses an LLM to generate precise, context‑aware responses.

Key Features

  • RAG Pipeline: Combines a retriever (ChromaDB) with a generator (LLaMA‑2) to produce evidence‑based answers.
  • Medical Corpus: Indexed a curated set of medical articles and PubMed abstracts.
  • User Interface: Simple web interface built with Streamlit for querying and displaying answers with source references.
  • Evaluation: Achieved 85% accuracy on a test set of 200 medical questions.

Technologies Used

Python LangChain ChromaDB LLaMA‑2 Streamlit

My Contributions

Designed the RAG pipeline, implemented the document ingestion and retrieval modules, integrated the LLM, and built the front‑end interface. Also conducted the accuracy evaluation and fine‑tuned prompts for medical domain.
